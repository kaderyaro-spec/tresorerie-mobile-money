Automatisation

📩 Lecture automatique des SMS

Faites remonter automatiquement les SMS (et notifications) de vos opérateurs : les transactions sont créées automatiquement, ou mises en attente de confirmation si la lecture est incertaine.

Sécurité : une opération n'est créée seule que si l'opérateur, le sens, le montant et l'identifiant de transaction sont lus correctement (l'identifiant évite tout doublon). Sinon, elle attend votre confirmation.

📨 Opérateurs par SMS (Orange, MTN, Moov)
  1. Installez une app gratuite de transfert SMS (ex. « SMS to URL Forwarder »).
  2. Règle : POST vers le lien ci-dessus.
  3. Corps : champ body = contenu du SMS (%text% ou {{ '{{message}}' }} selon l'app), sender = expéditeur.
  4. Filtrez sur les expéditeurs (Orange, MTN, Moov) pour ne transférer qu'eux.
🌊 Wave (par notification de l'app marchand)

Wave n'envoie pas de SMS : les transactions arrivent en notification dans l'app marchand. On les capte autrement :

  1. Installez une app gratuite de transfert de notifications (ex. « Notification Forwarder », ou MacroDroid / Tasker).
  2. Réglez-la pour capter les notifications de l'app Wave Business.
  3. Même règle : POST vers le lien ci-dessus, le texte de la notification dans le champ body, sender=Wave.

⚠️ Ce lien est secret : ne le partagez pas. En cas de doute, regénérez-le.
